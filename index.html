This is a simple website built to demonstrate authenticating with OAuth 2.0. The site implements an OAuth 2.0 Server Side flow without using 3rd party OAuth libraries. This demo is deployed on the Google Cloud Platform, and built with Python, HTML, and CSS.

Source Code: https://github.com/CoreySavage/OAuth2_Demo

How to use: To authenticate using OAuth 2.0, press the red button below and select the Google account you wish to use for your authentication. Authenticated is done through Google+, after authentication is complete you will be able to view some basic information about yourself provided by Google. No personal information is collected.
